# hardware-interface #

This module contains the STMicroelectronics proprietary hardware interfaces definitions.
It is part of the STMicroelectronics delivery for Android.

## Description ##

This module integrate proprietary hardware interfaces.
Please see the Android delivery release notes for more details.

## Documentation ##

* The [release notes][] provide information on the release.
[release notes]: https://wiki.st.com/stm32mpu-ecosystem-v5/wiki/STM32_MPU_OpenSTDroid_release_note_-_v5.1.0

## Containing ##

This module contains several files and directories.
* `copro`: hardware copro aidl interface definition and default implementation (read following [README](./copro/aidl/default/README.md) file for more details)
